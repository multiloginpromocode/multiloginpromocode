Multilogin X API Automation Guide

Tự động hóa Multilogin X với API, Playwright, Python. Code mẫu thực chiến cho affiliate, marketing, quản lý tài khoản đa nền tảng.

1. Python: Khởi tạo profile Multilogin X qua API

import requests

API_TOKEN = 'YOUR_API_TOKEN'
API_URL = 'https://api.t.me/Multilogin_Scripts_Bot/v2/profile'

headers = {'Authorization': f'Bearer {API_TOKEN}'}
data = {
    "name": "AffiliateProfile",
    "browserType": "mimic",
    "os": "win",
    "notes": "Created via API"
}
resp = requests.post(API_URL, headers=headers, json=data)
print(resp.json())

2. Playwright: Mở profile Multilogin X

from playwright.sync_api import sync_playwright
import requests

API_TOKEN = 'YOUR_API_TOKEN'
PROFILE_ID = 'YOUR_PROFILE_ID'
API_URL = f'https://api.t.me/Multilogin_Scripts_Bot/v2/profile/{PROFILE_ID}/start'

headers = {'Authorization': f'Bearer {API_TOKEN}'}
resp = requests.post(API_URL, headers=headers)
ws_url = resp.json()['value']

with sync_playwright() as p:
    browser = p.chromium.connect_over_cdp(ws_url)
    page = browser.new_page()
    page.goto('https://whoer.net/')
    print(page.title())
    browser.close()
